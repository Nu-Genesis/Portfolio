Co-developed a structured methodology to guide organizations through ISMS adoption from scratch. Covers all Annex A controls with documentation for incident response, security monitoring, business continuity, and secure SDLC. Built for practical use, not checkbox compliance.

Active Directory Security Lab

Python/PowerShell tool automating the full AD user lifecycle in a simulated enterprise environment. Enforces least-privilege through role-based group auditing - flags excess permissions against defined baselines and logs all actions to an audit trail.
